# Semantic Segmentation Web Application

A complete web application for training and testing semantic segmentation models using the Pascal VOC 2012 dataset.

## Features

- **Web Interface**: User-friendly interface for model training and prediction
- **Dataset Management**: Automatic dataset downloading and setup
- **Model Training**: Train semantic segmentation models with real-time progress tracking
- **Image Prediction**: Upload images and get segmentation results
- **Real-time Monitoring**: Live training logs and progress updates
- **Model Management**: Save and load trained models
